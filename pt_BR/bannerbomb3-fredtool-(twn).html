    Leitura Obrigatória

    Para fazer o dump do sistema DSiWare, nós executamos um exploit em uma falha na janela do Gerenciador de Dados do DSiWare no aplicativo Configuração do Console.

    Para fazer isso, usamos a chave de criptografia do seu sistema (movable.sed) para construir um backup de DSiWare que faz exploit do sistema para fazer um dump do aplicativo Conexões Nintendo DS para a raiz do SD.

    Assim que tiver um backup de DSiWare, um aplicativo exploitable de DSiWare pode ser inserido no DS Download Play, que pode ser usado para instalar um custom firmware.

    Essas instruções são APENAS para consoles tailandeses (indicados por um T no final da versão do sistema, ex. 11.16.0-39T)! Se o seu console for de qualquer outra região, escolha o outro método.

    O que você precisa

    • Seu arquivo movable.sed obtido por completar Seedminer
    • A versão mais recente do Luma3DS (o arquivo .zip do Luma3DS)
    • A última versão do b9stool (download direto)
    • A versão mais recente do Frogminer_save (Frogminer_save.zip)

    Seção I - Checando por CFW

    Como uma medida de segurança adicional, realizaremos uma verificação adicional de custom firmware. Isso é porque usar esse método quando custom firmware já está instalado tem o risco de danificar o console (renderizando-o sem métodos de recuperação como ntrboot).

    1. Desligue seu console
    2. Segure o botão (Select)
    3. Ligue o seu console enquanto ainda segura o botão (Select)
    4. Se a verificação foi bem-sucedida, você iniciará no Menu HOME e poderá prosseguir com este guia
    5. Desligue seu console

    Se você ver um menu de configuração ou o console é imediatamente desligado, você já possui CFW e continuar com estas instruções pode causar um BRICK no seu console! Siga Checando por CFW para atualizar o seu CFW existente.

    Seção II - Preparação

    1. Desligue seu console
    2. Insira o cartão SD no seu computador
    3. Copie boot.firm e boot.3dsx do .zip do Luma3DS para a raiz do seu cartão SD
      • A raiz do cartão SD refere-se ao diretório inicial no seu cartão SD onde você pode ver a pasta Nintendo 3DS, mas não está dentro dela
    4. Copie boot.nds (B9STool) do .zip para a raiz do seu cartão SD
    5. Copie a pasta private do .zip do Frogminer_save para a raiz do seu cartão SD
    6. Mantenha o seu cartão SD no seu computador - há mais coisas a fazer na próxima seção

    Seção III - BannerBomb3

    1. Abra o site DSIHaxInjector V2 no seu computador
    2. Sob o campo “Username”, digite qualquer nome alfanumérico (sem espaços ou caracteres especiais)
    3. Sob o campo “MovableSed”, envie seu movable.sed usando a opção “Browse…”
    4. Clique em “Build”
      • Aguarde alguns segundos para que o processo de compilação termine
    5. Na seção Build History à esquerda, digite o nome de usuário no campo “Filter Builds”
    6. Clique no primeiro resultado da pesquisa
      • Esse resultado deve ter a última marcação de tempo
    7. Clique no link “output_(nome).zip”
    8. Navegue até Nintendo 3DS -> <ID0> ->` em seu cartão SD
      • <ID0> é o nome da pasta de 32 caracteres que você copiou em Seedminer
      • <ID1> é uma pasta de 32 caracteres dentro da pasta <ID0>
      • Se você tiver múltiplas pastas `, siga as instruções [aqui](troubleshooting#bannerbomb3)
    9. Crie uma pasta chamada Nintendo DSiWare dentro do <ID1>
      • Se você já teve a pasta e já existe algum arquivo de backup de DSiWare existente (`. dentro) dentro, copie-os para seu PC e remova-os do seu cartão SD
    10. Abra o arquivo output_(nome) .zip e navegue até China_Taiwan -> output_setup
    11. Copie todos os seis arquivos .bin para a pasta Nintendo DSiWare
    12. Reinsira o cartão SD no seu console
    13. Ligue o seu console
    14. Abra as Configurações do Sistema no seu console
    15. Navegue até Gerenciador de Dados -> DSiWare -> Cartão SD -> (página 2)
    16. Observe que número está no aplicativo DSiWare da segunda página (será um número entre 1 e 6)
    17. Desligue seu console
    18. Insira o cartão SD no seu computador
    19. Navegue até Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare no seu cartão SD
    20. Abra o arquivo output_(nome) .zip e navegue até China_Taiwan -> output_hax
    21. Copie o arquivo .bin que é o mesmo número que estava na segunda página do menu do Gerenciador de DSiWare na pasta Nintendo DSiWare, e substitua quando solicitado
    22. Ligue o seu console
    23. Abra as Configurações do Sistema no seu console
    24. Navegue para Gerenciamento de dados -> DSiWare -> Cartão SD (imagem)
    25. Espere um pouco
      • O seguinte deve acontecer, nesta ordem: um pouco de progresso, um pequeno congelamento, a tela inferior fica roxa, a música parando, depois o console exibindo uma mensagem de erro e reiniciando
      • Se você não viu a tela roxa, então algo deu errado
    26. Agora você deve ter o arquivo 42383821.bin na raiz do seu cartão SD
    27. Navegue até Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare no seu cartão SD
    28. Apague todos os arquivos .bin da pasta Nintendo DSiWare

    Seção IV - Fredtool

    1. Abra o site DSIHaxInjector_new no seu computador
    2. Sob o campo “Username”, digite qualquer nome alfanumérico (sem espaços ou caracteres especiais)
      • Talvez você queira colocar um nome diferente para diferenciar do arquivo do BannerBomb3
    3. Sob o campo “DSiBin”, coloque seu arquivo 42383821.bin usando a primeira opção “Browse…”
    4. Sob o campo “MovableSed”, coloque seu arquivo movable.sed usando a segunda opção “Browse…”
    5. Sob o campo “InjectionTarget”, defina o alvo de injeção como DSdownloadplay (NÃO o memorypit)
    6. Clique em “Build”
      • Aguarde alguns segundos para que o processo de compilação termine
    7. Na seção Build History à esquerda, digite o nome de usuário no campo “Filter Builds”
    8. Clique no primeiro resultado da pesquisa
      • Esse resultado deve ter a última marcação de tempo
    9. Clique no link “output_(nome).zip”
    10. Navegue até Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare no seu cartão SD
    11. Copie o arquivo 484E4441.bin da pasta hax do arquivo DSiWare baixado (output_(nome).zip) para a pasta Nintendo DSiWare
    12. Reinsira o cartão SD no seu console
    13. Ligue o seu console
    14. Abra as Configurações do Sistema no seu console
    15. Navegue para Gerenciador de Dados -> DSiWare
    16. Na seção “Cartão SD”, selecione o aplicativo “Haxxxxxxxxxxx!”
    17. Selecione “Copy”, então selecione “OK”

    Seção V - Flipnote Exploit

    Nesta seção, você executará uma série de passos muito específicos dentro do Flipnote Studio que, quando executado corretamente, irá iniciar o b9sTool, o instalador do boot9strap (custom firmware).

    Caso você prefira um guia visual para esta seção, há um disponível aqui.

    1. Saia da Configuração do Console
    2. Inicie o Download Play no seu console (o ícone laranja com um 3DS nele)
    3. Selecione “Nintendo DS”
    4. Se o exploit foi bem sucedido, seu 3DS será carregado na versão japonesa do Flipnote Studio
    5. Complete o processo inicial do jogo até você chegar no menu principal
      • Selecione a opção da esquerda sempre que solicitado durante o processo de instalação
      • Se você encontrar um problema ao fazer essa seção, confira este guia de troubleshooting para seu erro
    6. Usando a touch-screen, selecione a caixa grande da esquerda, então selecione a caixa com um ícone do cartão SD
    7. Quando o menu carregar, selecione o ícone do rosto, então o ícone no canto inferior direito para continuar
    8. Aperte (X) ou (CIMA) no D-Pad dependendo do que for mostrado na tela superior
    9. Selecione o segundo botão no topo com um ícone de rolo de filme
    10. Role até o rolo “3/3” ser selecionado
    11. Toque na terceira caixa com a letra “A” nela
    12. Role à esquerda até o rolo “1/3” ser selecionado
    13. Toque na quarta caixa com a letra “A” nela
    14. Se o exploit foi bem sucedido, seu console irá abrir o b9sTool

    Seção VI - Instalando o boot9strap

    Nesta seção, você instalará custom firmware no seu console.

    1. Usando os botões direcionais, mova para “Install boot9strap”
      • Se você perder este passo, o sistema irá voltar para o Menu HOME ao invés de instalar o boot9strap e você precisará abrir o DS Download Play e reiniciar essa seção desde o início
    2. Pressione (A), depois pressione START e SELECT ao mesmo tempo para iniciar o processo
    3. Assim que for concluído e a tela inferior dizer “done.”, saia do b9sTool, depois desligue o seu console
      • Você pode ter que forçar o desligamento segurando o botão POWER
      • Se o seu console desligar quando você tentar ligá-lo, certifique-se de que você copiou boot.firm do .zip do Luma3DS para a raiz do seu cartão SD
      • Se você ver a tela de configuração do Luma, desligue o console e continue para a próxima seção

    Seção VII - Configuração do Luma3DS

    1. Aperte e segure (Select) e enquanto segura (Select), ligue seu console
    2. Seu console deve ter inicializado o menu de configuração do Luma3DS
      • O menu de configuração do Luma3DS são configurações para o custom firmware do Luma3DS. Muitas dessas configurações podem ser úteis para personalização ou depuração
      • Para o propósito deste guia, deixe essas opções nas configurações padrão (não marque ou desmarque nada)
      • Se o seu console desligar quando você tentar ligá-lo, certifique-se de que você copiou boot.firm do .zip do Luma3DS para a raiz do seu cartão SD
    3. Aperte (Start) para salvar e reiniciar

    Seção VIII - Restaurando o DS Download Play

    1. Navegue até Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare no seu cartão SD
    2. Copie o arquivo 484E4441.bin da pasta clean do arquivo DSiWare baixado (output_(nome).zip) para a pasta Nintendo DSiWare
    3. Reinsira o cartão SD no seu console
    4. Ligue o seu console
    5. Abra as Configurações do Sistema no seu console
    6. Navegue para Gerenciador de Dados -> DSiWare
    7. Na seção “Cartão SD”, selecione o aplicativo “Haxxxxxxxxxxx!”
    8. Selecione “Copy”, então selecione “OK”
    9. Saia da Configuração do Console
    10. Desligue seu console
