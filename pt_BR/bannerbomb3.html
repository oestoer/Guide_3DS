    Detalhes técnicos (opcional)

    Para iniciar o código personalizado, nós usamos um exploit no Gerenciamento de Dados DSiWare no aplicativo Configurações do Sistema.

    Para fazer isso, usamos a chave de criptografia do seu sistema (movable.sed) para construir um backup de DSiWare que se aproveita do exploit do sistema.

    Para obter uma explicação mais técnica, veja aqui.

    Notas de Compatibilidade

    Essas instruções funcionam nas versões de consoles EUA, Europa, Japão e Coréia, indicadas pelas letras U, E, J ou K após a versão do sistema.

    Se você tiver um console tailandês (indicado por um T após a versão do sistema), siga esta página ao invés desta.

    O que você precisa

    • Seu arquivo movable.sed de completar Seedminer
    • A versão mais recente do BannerBomb3 (Release_v3.0.zip, download direto)

    Seção I - Preparação

    Nesta seção, você irá copiar os arquivos necessários para acionar o exploit BannerBomb3 no cartão SD do seu console.

    1. Desligue seu console
    2. Insira o cartão SD no seu computador
    3. Abra o Bannerbomb3 Injector no seu computador
    4. Envie seu movable.sed usando a opção “Choose File”
    5. Clique em “Build and Download”
      • Isto irá baixar um DSiWare chamado F00D43D5.bin, que estará contido em um arquivo zip (BannerBomb3.zip)
    6. Copie bb3.bin do Release_v3.0.zip para a raiz do seu cartão SD
      • A raiz do cartão SD refere-se ao diretório inicial no seu cartão SD onde você pode ver a pasta Nintendo 3DS, mas não está dentro dela
      • Este arquivo não precisa ser aberto ou extraído
    7. Copie Launcher.dat da pasta para seu modelo e região dentro do Release .zip para a raiz do seu cartão SD
    8. Navegue até Nintendo 3DS -> <ID0> ->` em seu cartão SD
      • <ID0> é o nome da pasta de 32 caracteres que você copiou em Seedminer
      • <ID1> é uma pasta de 32 caracteres dentro da pasta <ID0>
      • Se você tiver múltiplas pastas `, siga as instruções [aqui](troubleshooting#bannerbomb3) ![](https://3ds.hacks.guide/images/screenshots/bb3/dsiware-location-1.png){: .notice--info}
    9. Crie uma pasta chamada Nintendo DSiWare dentro do <ID1>
      • Se você já teve a pasta e já existe algum arquivo de backup de DSiWare existente (`. dentro) dentro, copie-os para seu PC e remova-os do seu cartão SD
    10. Copie o arquivo F00D43D5.bin do BannerBomb3.zip para a pasta Nintendo DSiWare

    Seção II - BannerBomb3

    Nesta seção, você irá acionar o exploit BannerBomb3 usando o menu de Gerenciamento de DSiWare e usá-lo para instalar o exploit menuhax67.

    1. Reinsira o cartão SD no seu console
    2. Ligue o seu console
    3. Abra as Configurações do Sistema no seu console
    4. Navegue para Gerenciamento de dados -> DSiWare -> Cartão SD (imagem)
      • Seu console deve mostrar o menu multihax do BB3
    5. Use os botões direcionais para navegar e pressione o botão (A) para selecionar “Install *hax”
      • Seu console irá reiniciar automaticamente
    6. Desligue seu console
