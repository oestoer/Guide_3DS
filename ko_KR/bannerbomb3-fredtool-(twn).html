    BannerBomb3 + Fredtool (타이완)

    영어로 지원을 요청하고 싶으시면 Nintendo Homebrew Discord 서버에 도움을 요청해 주세요.
    이 가이드가 도움이 되었다면, 기부해 주시면 감사하겠습니다.

    중요

    시스템 DSiWare를 추출하기 위해, DSiWare 데이터 관리 설정 창의 취약점을 악용합니다.

    이 가이드는 시스템의 암호화 키 (movable.sed)를 이용해 DSiWare 백업을 만들어, 시스템이 DSi 인터넷 설정 앱을 SD 카드 루트에 추출하게 합니다.

    DSiWare 백업이 있으면, DS 다운로드 플레이에 취약한 DSiWare 타이틀을 주입할 수 있게 되어, 이를 이용해서 커스텀 펌웨어를 설치할 수 있습니다.

    이 페이지는 타이완 콘솔만에 해당됩니다! (시스템 버전 마지막에 T가 붙음, 예: 11.16.0-39T) 만약 다른 지역의 콘솔을 갖고 계시다면, 다른 방법을 선택해 주세요.

    준비물

    • Seedminer에서 받은 movable.sed 파일
    • 최신 버전의 Luma3DS (.zip 파일)
    • 최신 버전의 b9stool (직접 다운로드)
    • 최신 버전의 Frogminer_save (Frogminer_save.zip)

    섹션 I - CFW 확인

    여기서 콘솔의 안전을 위해 커스텀 펌웨어가 설치되어 있는지 확인합니다. 이 페이지의 방법은, 이미 커스텀 펌웨어가 설치되어 있다면 콘솔을 벽돌 상태로 남길 가능성이 있습니다. (이럴 시 해결 방법은 ntrboot 같은 방법으로 해결할 수 밖에 없습니다.)

    1. 콘솔의 전원을 꺼 주세요
    2. (Select) 버튼을 길게 눌러 주세요
    3. (Select) 버튼을 계속 눌러 있는 상태에서 콘솔의 전원을 켜 주세요
    4. 만약 HOME 메뉴가 켜지면, 이 확인은 성공적이고, 가이드를 진행하면 됩니다
    5. 콘솔의 전원을 꺼 주세요

    만약 설정 메뉴가 보이거나 콘솔의 전원이 바로 종료된다면, 이 콘솔에은 이미 커펌이 설치되어 있고, 이 작업을 진행하면 콘솔이 벽돌 될수가 있습니다! 이미 존재하는 CFW를 업데이트 할 수 있도록 CFW 확인을 진행해 주세요.

    섹션 II - 준비 작업

    1. 콘솔의 전원을 꺼 주세요
    2. SD 카드를 컴퓨터에 삽입해 주세요
    3. Luma3DS .zip안에 있는 boot.firm과 boot.3dsx를 SD 카드의 루트에 복사해 주세요
      • SD 카드의 루트는 SD 카드를 열 때 보이는 디렉토리이고, Nintendo 3DS 폴더가 보이지만 그 안에 들어가 있지 않은 상태의 폴더입니다
    4. release .zip안에 있는 boot.nds (B9STool)를 SD 카드의 루트에 복사해 주세요
    5. SD 카드의 루트로 Frogminer_save .zip안에 압축이 되있는 private 폴더를 복사해 주세요
    6. SD 카드는 컴퓨터에서 빼지 마세요. 다음 섹션에서 할 일이 더 있습니다

    섹션 III - BannerBomb3

    1. DSIHaxInjector V2 사이트를 컴퓨터에서 여세요
    2. “Username”에 영어/숫자를 사용한 이름을 아무거나 입력해 주세요 (스페이스나 특수 문자는 사용 불가)
    3. “MovableSed” 란에, “Browse…” 옵션을 이용해 movable.sed 파일을 업로드해 주세요
    4. “Build”를 클릭해 주세요
      • 빌드 과정이 완료될 때까지 잠시 기다려 주세요
    5. 왼쪽의 Build History 영역에 유저네임을 입력해 주세요
    6. 첫 번째 검색 결과를 선택해 주세요
      • 날짜가 가장 최신이어야 합니다
    7. “output_(name).zip” 링크를 클릭하여 다운로드해 주세요
    8. SD 카드의 Nintendo 3DS -> <ID0> -> <ID1> 폴더로 이동해 주세요
      • <ID0>은 Seedminer에서 복사한 32글자 폴더명 입니다
      • <ID1>은 <ID0> 안에 있는 32글자 폴더명입니다
      • 여러 개의 <ID1> 폴더가 보인다면, 이 안내를 따라주세요
    9. <ID1> 폴더에 Nintendo DSiWare 폴더를 생성해 주세요
      • 만약 그 폴더가 있고 내부에 DSiWare 백업 파일(<8-character-id>.bin)이 있다면, PC로 모두 복사해 두고 SD 카드에서 삭제해 주세요
    10. output_(name) .zip 파일을 열고 China_Taiwan -> output_setup으로 이동해주세요
    11. 6개의 .bin 파일을 모두 Nintendo DSiWare 폴더로 복사해 주세요
    12. SD 카드를 콘솔에 다시 삽입해 주세요
    13. 콘솔의 전원을 켜 주세요
    14. 콘솔에서 본체 설정을 열어 주세요
    15. 데이터 관리 -> DSiWare -> SD 카드 -> 2페이지로 이동해 주세요
    16. 2페이지에 있는 DSiWare 타이틀의 숫자를 메모해 주세요 (1부터 6까지 중 하나일 것입니다)
    17. 콘솔의 전원을 꺼 주세요
    18. SD 카드를 컴퓨터에 삽입해 주세요
    19. SD 카드의 Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare 폴더로 이동해 주세요
    20. output_(name) .zip 파일을 열어 China_Taiwan -> output_hax로 이동해 주세요
    21. DSiWare 관리 메뉴의 2페이지에 있던 것과 같은 숫자를 가진 .bin 파일을 Nintendo DSiWare 폴더로 복사해주세요. 파일이 이미 있다면 덮어씌워 주세요
    22. 콘솔의 전원을 켜 주세요
    23. 콘솔에서 본체 설정을 열어 주세요
    24. 데이터 관리 -> DSiWare -> SD 카드로 들어가주세요 (사진)
    25. 잠시 기다려 주세요
      • 다음이 순서대로 발생할 것입니다: 진행도 바퀴가 표시되었다가 콘솔이 멎고, 하단 스크린이 보라색으로 바뀌며 음악이 멈추고, 그 후에 콘솔에서 오류 메시지를 표시한 후 재부팅 됩니다
      • 보라색 화면이 뜨지 않았다면, 뭔가가 잘못된 겁니다
    26. 이제 SD 카드 최상위 폴더에 42383821.bin 파일이 있을 겁니다
    27. SD 카드의 Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare 폴더로 이동해 주세요
    28. Nintendo DSiWare 폴더의 모든 .bin 파일을 삭제해 주세요
    섹션 IV - Fredtool
    1. 컴퓨터에서 DSIHaxInjector_new 웹사이트를 열어 주세요
    2. “Username”에 영어/숫자를 사용한 이름을 아무거나 입력해 주세요 (스페이스나 특수 문자는 사용 불가)
      • BannerBomb3의 출력과 차이를 두려면 다른 이름을 넣어야 할 수 있습니다
    3. “DSIBin” 란에, “Browse…“옵션을 이용해 42383821.bin 파일을 업로드해 주세요
    4. “MovableSed” 란에, “Browse…” 옵션을 이용해 movable.sed 파일을 업로드해 주세요
    5. “InjectionTarget” 란에서, 주입 목표를 DSdownloadplay(memorypit 아님)로 설정해 주세요
    6. “Build”를 클릭해 주세요
      • 빌드 과정이 완료될 때까지 잠시 기다려 주세요
    7. 왼쪽의 Build History 영역에 유저네임을 입력해 주세요
    8. 첫 번째 검색 결과를 선택해 주세요
      • 날짜가 가장 최신이어야 합니다
    9. “output_(name).zip” 링크를 클릭하여 다운로드해 주세요
    10. SD 카드의 Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare 폴더로 이동해 주세요
    11. 다운로드한 DSiWare 압축 파일(output_(이름).zip)에서, hax 폴더 안에 있는 484E4441.bin 파일을 Nintendo DSiWare 폴더로 복사해 주세요
    12. SD 카드를 콘솔에 다시 삽입해 주세요
    13. 콘솔의 전원을 켜 주세요
    14. 콘솔에서 본체 설정을 열어 주세요
    15. 데이터 관리 -> DSiWare 메뉴를 열어 주세요
    16. “SD 카드” 섹션에서 “Haxxxxxxxxx!” 타이틀을 선택해 주세요
    17. “복사”를 선택한 후, “OK”를 선택해 주세요

    섹션 V - Flipnote 익스플로잇

    이 섹션에서는, boot9strap (커스텀 펌웨어) 설치 도구를 실행 할 수 있도록 Flipnote Studio에서 여러 가지 단계를 진행 합니다.

    만약 이 섹션의 비주얼 가이드를 선호한다면, [여기서] (https://zoogie.github.io/web/flipnote_directions/) 그 중 하나를 보실 수가 있습니다.

    1. 본체 설정을 종료해 주세요
    2. 콘솔에서 다운로드 플레이 앱(3DS가 그려져 있는 주황색 아이콘)을 실행해 주세요
    3. “Nintendo DS”를 선택해 주세요
    4. 만약 익스플로잇이 성공적이었다면, 콘솔은 JPN 버전의 Flipnote Studio를 실행할 겁니다
    5. 메인 메뉴로 갈 때까지의 초기 설정을 완료하세요
      • 만약 설정 중 선택이 표시될 시는 왼쪽을 선택해 주세요
      • If you encounter an issue while doing this section, check this troubleshooting guide for your issue
    6. 터치 스크린을 사용해서, 오른쪽의 큰 버튼을 선택하고, 그 후 SD 카드 아이콘이 붙은 버튼을 선택해 주세요
    7. 메뉴가 로드되면, 얼굴 아이콘을 선택하고, 오른쪽 아래 버튼을 눌러서 진행해 주세요
    8. 위 화면에 나오는 것에 따라, (X)나 십자 패드의 (위)를 눌러 주세요
    9. 아래 화면의 상단 두 번째에 있는, 필름 아이콘을 선택해 주세요
    10. “3/3”이 선택될 때까지 오른쪽으로 스크롤해 주세요
    11. “A”가 들어가 있는 세 번째 버튼을 눌러 주세요
    12. “1/3”이 선택될 때까지 왼쪽으로 스크롤해 주세요
    13. “A”가 들어가 있는 네 번째 버튼을 눌러 주세요
    14. 취약점이 성공적이었다면 b9sTool을 실행했을 것입니다

    섹션 VI - boot9strap 설치

    이 섹션에서는, 커스텀 펌웨어를 설치할 것입니다.

    1. 십자 패드를 사용해서, “Install boot9strap”으로 이동해 주세요
      • If you miss this step, the system will exit to HOME Menu instead of installing boot9strap and you will need to start over from the previous section
    2. (A) 버튼을 누른 후, START와 SELECT를 동시에 눌러 작업을 실행해 주세요
    3. 완료되고 아래 화면에 “done.”이라고 표시되면, b9sTool를 종료하고 콘솔의 전원을 꺼 주세요
      • 전원버튼을 계속 눌러 강제종료를 해야 할 수도 있습니다
      • 만약 콘솔이 부팅시에 꺼진다면 Luma3DS .zip 파일 안에 압축 되있는 boot.firm을 SD 카드의 루트로 복사 했는지 확인해 주세요
      • 만약 Luma3DS 구성 메뉴가 표시되면, 콘솔의 전원을 끄고 다음 섹션으로 넘어가 주세요

    섹션 VII - Luma3DS 구성

    1. (Select) 를 길게 누르고, 이 상태에서 콘솔의 전원을 켜 주세요
    2. 콘솔은 Luma3DS 구성 메뉴로 부팅되었을 겁니다
      • Luma3DS 구성 메뉴는 Luma3DS 커스텀 펌웨어의 설정들 입니다. 이 설정들은 디버깅 또는 맞춤화의 유용합니다
      • 일단 이 가이드에서는 모든 설정을 기본으로 남겨 두세요 (켜기 또는 끄기를 누르지 마세요)
      • 만약 콘솔이 부팅시에 꺼진다면 Luma3DS .zip 파일 안에 압축 되있는 boot.firm을 SD 카드의 루트로 복사 했는지 확인해 주세요
    3. (Start)를 눌러 저장하고 콘솔을 다시 시작해 주세요

    섹션 VIII - DS 다운로드 플레이 복구

    1. SD 카드의 Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare 폴더로 이동해 주세요
    2. 다운로드한 DSiWare 압축 파일(output_(이름).zip)에서, clean 폴더 안에 있는 484E4441.bin 파일을 Nintendo DSiWare 폴더로 복사해 주세요
    3. SD 카드를 콘솔에 다시 삽입해 주세요
    4. 콘솔의 전원을 켜 주세요
    5. 콘솔에서 본체 설정을 열어 주세요
    6. 데이터 관리 -> DSiWare 메뉴를 열어 주세요
    7. “SD 카드” 섹션에서 “Haxxxxxxxxx!” 타이틀을 선택해 주세요
    8. “복사”를 선택한 후, “OK”를 선택해 주세요
    9. 본체 설정을 종료해 주세요
    10. 콘솔의 전원을 꺼 주세요
