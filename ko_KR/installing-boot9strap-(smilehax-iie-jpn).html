    Installing boot9strap (smilehax-IIe-JPN)

    영어로 지원을 요청하고 싶으시면 Nintendo Homebrew Discord 서버에 도움을 요청해 주세요.
    이 가이드가 도움이 되었다면, 기부해 주시면 감사하겠습니다.

    목차

    • 준비물
    • 진행 방법
      • 섹션 I - 준비 작업
      • 섹션 II - smilehax
      • 섹션 III - boot9strap 설치하기
    • 마무리 단계 로 계속합니다

    This set of instructions does not support the USA and EUR versions of SmileBASIC. The guide for those versions can be found here.

    기술적 상세 정보 (선택 사항)

    smilehax-IIe is an exploit for the game “SmileBASIC” made by zoogie, compatible with system versions 9.0.0 through 11.17.0 for USA/EUR/JPN region consoles.

    준비물

    • The game “SmileBASIC” installed on your console
      • 이 게임을 다운로드 한 적이 있다면, 다시 다운로드 할 수 있습니다
      • Your SD card must be inserted in your console to install SmileBASIC
    • The latest release of smilehax-IIe (direct download)
    • 최신 버전의 SafeB9SInstaller (직접 다운로드)
    • 최신 버전의 boot9strap (직접 다운로드)
    • 최신 버전의 nimdsphax (직접 다운로드)
    • 최신 버전의 Luma3DS (.zip 파일)
    • 최신 버전의 HBL otherapp 파일들 (직접 다운로드)

    진행 방법

    섹션 I - 준비 작업

    1. Make sure SmileBASIC is updated to the latest version
    2. 콘솔의 전원을 꺼 주세요
    3. SD 카드를 컴퓨터에 삽입해 주세요
    4. Luma3DS .zip안에 있는 boot.firm과 boot.3dsx를 SD 카드의 루트에 복사해 주세요
      • SD 카드의 루트는 SD 카드를 열 때 보이는 디렉토리이고, Nintendo 3DS 폴더가 보이지만 그 안에 들어가 있지 않은 상태의 폴더입니다
    5. SD 카드이 루트에 boot9strap 라는 이름으로 새 폴더를 생성해 주세요
    6. SD 카드에 있는 /boot9strap/ 폴더에 boot9strap .zip 파일 안에 압축되있는 boot9strap.firm과 boot9strap.firm.sha을 복사해 주세요
    7. SafeB9SInstaller .zip의 SafeB9SInstaller.bin을 SD 카드의 루트로 복사해 주세요
    8. Copy new3ds_JPN.bin or old3ds_JPN.bin (depending on your model) from otherapps_with_CfgS_11.17.zip to the root of your SD card, then rename the payload to otherapp.bin
      • .bin 확장자는 보이지 않는 이상 추가하지 말아 주세요
    9. Transfer SH2E-JPN.300.wav from the Release_sh2e .zip to a device that can play .wav files, for example a phone
    10. 콘솔의 지역에 해당한 petitcom .icn 파일을 SD 카드의 루트에 복사해 주세요
    11. SD 카드의 루트에 3ds 폴더가 없을 경우 폴더를 만들어 주세요
      • 이 폴더에는 홈브류 에플리케이션과 데이터가 보관되어 있으며, 콘솔이 자동으로 생성하는 Nintendo 3DS 폴더와 다른 폴더입니다
    12. nimdsphax .zip안에 있는 nimdsphax 폴더를 SD 카드의 /3ds/ 폴더에 복사해 주세요
    13. Navigate to Nintendo 3DS -> <ID0> -> <ID1> -> title -> 0004000e
    14. Rename the 00117200 folder to _00117200
    15. SD 카드를 콘솔에 다시 삽입해 주세요
    16. 콘솔의 전원을 켜 주세요

    섹션 II - smilehax

    1. SmileBASIC을 실행해 주세요
      • DO NOT update the game if prompted
    2. “Publish/Download Projects” 를 선택해 주세요
    3. Select the “Download (Receive) using Public Key” option
    4. “Yes” 버튼을 눌러 주세요
    5. Enter the Public Key “NJEQK3A4” using the on-screen keyboard
    6. Press the “Yes” button to confirm the download of PMODEM141
    7. Press “OK”, then “Back”, then “Browse Projects”
    8. Select PMODEM141, then press “Select File”
    9. Select PMODEM141.PRG, then press “OK”, then “Yes” to execute it
    10. Select RECEIVE FILE SIMPLEX
    11. Prepare the device you use to play the .wav file
    12. Keep your console about a foot from the .wav file playing devices’ speaker and use a reasonable (not loud) volume to play the .wav
    13. Now select 300BPS, then play the .wav file
    14. When it finishes, press “Yes”, then press “OK”
    15. START 버튼을 눌러 주세요
    16. Select the SH2E-xxx.TXT file, then press OK, then “Yes” to execute it
    17. 콘솔은 Homebrew Launcher로 부팅되었을 겁니다
    18. 홈브류 목록에서 nimdsphax를 실행해 주세요
    19. 익스플로잇이 성공적이었다면 SafeB9SInstaller로 부팅되었을 것입니다
      • 만약 콘솔에 적색 화면이 표시되면, 전원 버튼을 길게 눌러 콘솔의 전원을 끄고, 이 섹션을 다시 시도해 주세요

    섹션 III - boot9strap 설치하기

    이 섹션에서는, 커스텀 펌웨어를 설치할 것입니다.

    1. 메시지가 나타나면, boot9strap을 설치하기 위하여 화면에 주어진 키 조합을 입력해 주세요
      • 아래 화면의 단계가 붉은 텍스트로 표시되고 키 콤보 입력을 요구하지 않는다면, 이 문제 해결 가이드를 참고해 주세요
    2. 작업이 완료되면, (A) 를 눌려 콘솔을 재부팅해 주세요
    3. 콘솔은 Luma3DS 구성 메뉴로 부팅되었을 겁니다
      • Luma3DS 구성 메뉴는 Luma3DS 커스텀 펌웨어의 설정들 입니다. 이 설정들은 디버깅 또는 맞춤화의 유용합니다
      • 일단 이 가이드에서는 모든 설정을 기본으로 남겨 두세요 (켜기 또는 끄기를 누르지 마세요)
      • 만약 콘솔이 부팅시에 꺼진다면 Luma3DS .zip 파일 안에 압축 되있는 boot.firm을 SD 카드의 루트로 복사 했는지 확인해 주세요
    4. (Start)를 눌러 저장하고 콘솔을 다시 시작해 주세요

    이제 콘솔은 Luma3DS를 기본으로 부팅 됩니다.

    • Luma3DS는 보통 HOME 메뉴와 똑같이 보입니다. 만약 콘솔이 HOME 메뉴로 부팅되었다면, 커스텀 펌웨어가 설치되어 있습니다.
    • 다음 페이지에서는 커펌에 유용한 홈브류를 설치해 콘솔 준비를 완료할 것 입니다.
