    Flash de ntrboot (plusieurs 3DS)

    Pour un support en Anglais, demander de l'aide sur le Discord Nintendo Homebrew.
    Si vous appréciez ce guide, nous acceptons les Dons.

    Lecture requise

    Before proceeding, ensure you have read all of the information on ntrboot.

    This method requires temporary access to a second 3DS family console that is already running boot9strap. Votre linker n’a pas besoin de fonctionner sur aucune des deux 3DS.

    Notez que dans de rares circonstances, il est possible que pendant le processus de flash sur un linker contrefait (ou un clone) celui-ci brique et devienne inutilisable de manière permanente. C’est peu probable, mais néanmoins, seuls les linkers originaux listés sont pris en charge. To reduce the chance of receiving a counterfeit card, it is recommended that you use a reputable site to buy your flashcart (such as NDS Card).

    Ce dont vous avez besoin

    • Votre linker compatible ntrboot 
    • Two 3DS family consoles
      • The source 3DS: the 3DS family console that is already running boot9strap
      • The target 3DS: the 3DS family console on stock firmware
    • La version v1.3 de boot9strap-ntr (téléchargement direct)
    • La dernière version de ntrboot_flasher (ntrboot_flasher.firm)

    Instructions

    Section I - Préparatifs

    1. Éteignez la 3DS source
    2. Insérez la carte SD de la 3DS source dans votre ordinateur
    3. Créez un dossier appelé ntrboot à la racine de votre carte SD
    4. Copiez boot9strap_ntr.firm et boot9strap_ntr.firm.sha depuis le fichier .zip de boot9strap ntr vers le dossier /ntrboot/ sur votre carte SD
    5. Copiez ntrboot_flasher.firm dans le dossier /luma/payloads/ sur la carte SD de la 3DS source
    6. Réinsérez la carte SD de la 3DS source dans la 3DS source
    7. Insérez votre linker DS / DSi compatible ntrboot dans la 3DS source

    Section II - Flasher ntrboot

    1. Démarrez le chainloader de Luma3DS en maintenant (Start) pendant le démarrage de la 3DS source
    2. Sélectionnez “ntrboot_flasher”
    3. Lisez l’avertissement de l’écran rouge
    4. Appuyez sur (A) pour continuer
    5. Sélectionnez votre linker Si vous ne voyez pas votre linker dans la liste en haut, lisez l’écran du bas pour plus d’informations sur chaque option
    6. Selectionnez “Dump Flash”
    7. Attendez que le processus soit terminé
    8. Appuyez sur (A) pour continuer
    9. Appuyez sur (A) pour revenir au menu principal
    10. Selectionnez “Inject Ntrboot”
    11. Appuyez sur (A) pour ntrboot retail
    12. Attendez que le processus soit terminé
    13. Appuyez sur (A) pour revenir au menu principal
    14. Appuyez sur (B) pour éteindre la 3DS source
