    Lecture requise

    Pour extraire un DSiWare système, nous exploitons une faille dans l’onglet de gestion des données DSiWare présent dans les paramètres système.

    Pour ce faire, nous utilisons la clé de chiffrement de votre console (movable.sed) pour créer une sauvegarde DSiWare qui exploite le système pour dumper l’application DSi Internet Settings à la racine SD.

    Une fois que vous avez une sauvegarde DSiWare, un titre DSiWare exploitable peut être injecté dans Téléchargement DS, qui peut être utilisé pour installer un custom firmware.

    Ces instructions sont destinées aux consoles taïwanaises UNIQUEMENT (indiqué par un T à la fin de la version du système, par exemple 11.16.0-39T) ! Si votre console provient d’une autre région, choisissez l’autre méthode.

    Ce dont vous avez besoin

    • Your movable.sed file completing Seedminer
    • La dernière version de Luma3DS (le fichier `.zip’ de Luma3DS)
    • The latest release of b9stool (direct download)
    • La dernière version de Frogminer_save (Frogminer_save.zip)

    Section I - Vérification de la présence d’un CFW

    Comme mesure de sécurité supplémentaire, nous effectuerons une vérification supplémentaire de la présence d’un custom firmware. Ceci est dû au fait que l’utilisation de cette méthode lorsqu’un custom firmware est déjà installé à un risque de bricker la console (ce qui la rendrait inutilisable sans méthodes de récupération comme ntrboot).

    1. Éteignez votre console
    2. Maintenez le bouton (Select) enfoncé
    3. Power on your console while still holding the (Select) button
    4. Si la vérification a réussi, vous démarrerez sur le menu HOME et vous pourrez continuer avec ce guide
    5. Éteignez votre console

    If you see a configuration menu or the console immediately powers off, you already have CFW, and continuing with these instructions may BRICK your console! Suivez Vérification de la présence d’un CFW pour mettre à jour votre CFW existant.

    Section II - Préparatifs

    1. Éteignez votre console
    2. Insérez votre carte SD dans votre ordinateur
    3. Copiez boot.firm et boot.3dsx depuis Luma3DS .zipvers la racine de votre carte SD
      • La racine de la carte SD représente le répertoire initial de la carte SD où vous pouvez voir le dossier Nintendo 3DS, mais ce n’est pas à l’intérieur de ce dernier
    4. Copiez boot.nds (B9STool) de l’archive .zip à la racine de votre carte SD
    5. Copiez le dossier private du fichier .zip de Frogminer_save vers la racine de votre carte SD
    6. Gardez votre carte SD dans votre ordinateur - Il y a d’autres choses à faire dans la prochaine section

    Section III - BannerBomb3

    1. Ouvrez le site Web DSIHaxInjector V2 sur votre ordinateur
    2. Dans le champ “Username”, entrez n’importe quel nom alphanumérique (sans espaces ni caractères spéciaux)
    3. Sous le champ “MovableSed”, uploadez votre movable.sed en utilisant l’option “Choisir un fichier”
    4. Cliquez sur “Build”
      • Attendez quelques secondes que le processus de compilation soit terminé
    5. Dans la section Historique des builds à gauche, tapez le nom d’utilisateur dans le champ “Filter builds…”
    6. Cliquez sur le premier résultat de recherche
      • Ce résultat devrait avoir l’horodatage le plus récent
    7. Cliquez sur le lien “output_(name).zip”
    8. Naviguez vers Nintendo 3DS -> <ID0> -> <ID1> sur votre carte SD
      • <ID0> est le nom du dossier à 32 lettres que vous avez copié dans Seedminer
      • <ID1> est un dossier de 32 caractères à l’intérieur de <ID0>
      • Si vous avez plusieurs dossiers <ID1>, suivez les instructions ici
    9. Créez un dossier nommé Nintendo DSiWare à l’intérieur de <ID1>
      • Si vous aviez déjà le dossier et qu’il y a des fichiers de sauvegarde DSiWare existants (<8-character-id>. in) à l’intérieur, copiez-les sur votre PC et retirez-les de votre carte SD
    10. Ouvrez le fichier output_(name) .zip et accédez à China_Taiwan -> output_setup
    11. Copiez les six fichiers “.bin” dans le dossier “Nintendo DSiWare”
    12. Réinsérez votre carte SD dans votre console
    13. Power on your console
    14. Launch System Settings on your console
    15. Accédez à Gestion des données -> DSiWare -> Carte SD -> (page 2)
    16. Prenez note du numéro sur le titre DSiWare sur la deuxième page (ce sera un numéro entre 1 et 6)
    17. Éteignez votre console
    18. Insérez votre carte SD dans votre ordinateur
    19. Naviguez vers Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare sur votre carte SD
    20. Ouvrez le fichier output_(name) .zip et accédez à China_Taiwan -> output_hax
    21. Copiez le fichier .bin portant le même numéro que celui qui se trouvait sur la deuxième page du menu de Gestion des Nintendo DSiWare dans le dossier Nintendo DSiWare et écrasez-le lorsque vous y êtes invité
    22. Power on your console
    23. Launch System Settings on your console
    24. Navigate to Data Management -> DSiWare -> SD Card (image)
    25. Attendez un peu
      • The following should happen, in this order: A progress swirl, a short freeze, the bottom screen turning purple, the music stopping, then the console showing an error message and rebooting
      • Si vous n’avez pas vu l’écran violet, alors quelque chose s’est mal passé
    26. Vous devriez maintenant avoir le fichier 42383821.bin à la racine de votre carte SD
    27. Naviguez vers Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare sur votre carte SD
    28. Supprimez tous les fichiers .bin du dossier Nintendo DSiWare

    Section IV - Fredtool

    1. Ouvrez le site Web DSIHaxInjector_new sur votre ordinateur
    2. Dans le champ “Username”, entrez n’importe quel nom alphanumérique (sans espaces ni caractères spéciaux)
      • Vous devriez mettre un nom différent pour le différencier de la sortie de BannerBomb3
    3. Sous le champ “DSiBin”, uploadez votre fichier 42383821.bin en utilisant la première option “Choisir un fichier”
    4. Sous le champ “MovableSed”, uploadez votre fichier movable.sed en utilisant la deuxième option “Choisir un fichier”
    5. Sous le champ “InjectionTarget”, définissez la cible d’injection sur DSdownloadplay (PAS memorypit)
    6. Cliquez sur “Build”
      • Attendez quelques secondes que le processus de compilation soit terminé
    7. Dans la section Historique des builds à gauche, tapez le nom d’utilisateur dans le champ “Filter builds…”
    8. Cliquez sur le premier résultat de recherche
      • Ce résultat devrait avoir l’horodatage le plus récent
    9. Cliquez sur le lien “output_(name).zip”
    10. Naviguez vers Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare sur votre carte SD
    11. Copiez le fichier 484E4441.bin du dossier hax de l’archive DSiWare téléchargée (output_(name).zip) dans le dossier Nintendo DSiWare
    12. Réinsérez votre carte SD dans votre console
    13. Power on your console
    14. Launch System Settings on your console
    15. Naviguez versGestion des données -> DSiWare
    16. Sous la section “Carte SD”, sélectionnez le titre “Haxxxxxxxxx!”
    17. Sélectionnez “Copier”, puis sélectionnez “OK”

    Section V - Exploit Flipnote

    Dans cette section, vous allez effectuer une série d’étapes très spécifiques dans Flipnote Studio qui, une fois effectuées correctement, déclencheront le lancement de b9sTool, l’installateur de boot9strap (custom firmware).

    Si vous préférez un guide visuel pour cette section, un tel guide est disponible ici.

    1. Quittez Paramètres de la console
    2. Launch Download Play on your console (the orange icon with a 3DS on it)
    3. Sélectionnez “Nintendo DS”
    4. Si l’exploit a réussi, votre 3DS devrait avoir démarré la version JPN de Flipnote Studio
    5. Terminez le processus de configuration initial de Flipnote Studio jusqu’à ce que vous atteigniez le menu principal
      • Sélectionnez l’option de gauche chaque fois que vous y êtes invité durant le processus de configuration
      • If you encounter an issue while doing this section, check this troubleshooting guide for your issue
    6. En utilisant l’écran tactile, sélectionnez la grande case à gauche, puis sélectionnez la boîte avec une icône de carte SD
    7. Une fois le menu lancé, sélectionnez l’icône visage, puis le bouton en bas à droite pour continuer
    8. Appuyez sur (X) ou (Haut) sur la croix directionnelle en fonction de ce qui est affiché sur l’écran supérieur
    9. Sélectionnez le deuxième bouton en haut avec une icône de bobine de film
    10. Défilez vers la droite jusqu’à ce que la bobine “3/3” soit sélectionnée
    11. Touchez la troisième case contenant la lettre “A”
    12. Défilez vers la gauche jusqu’à ce que la bobine “1/3” soit sélectionnée
    13. Touchez la quatrième case contenant la lettre “A”
    14. If the exploit was successful, your console will have loaded b9sTool

    Section VI - Installation de boot9strap

    In this section, you will install custom firmware onto your console.

    1. Séléctionnez “Install boot9strap” en utilisant la croix directionnelle
      • If you miss this step, the system will exit to HOME Menu instead of installing boot9strap and you will need to start over from the previous section
    2. Appuyez sur (A), puis appuyez sur START et SELECT en même temps pour démarrer l’installation
    3. Once completed and the bottom screen says “done.”, exit b9sTool, then power off your console
      • Vous devrez peut-être forcer l’arrêt de la console en maintenant le bouton Power enfoncé
      • If your console shuts down when you try to power it on, ensure that you have copied boot.firm from the Luma3DS .zip to the root of your SD card
      • If you see the Luma Configuration screen, power off your console and continue to the next section

    Section VII - Luma3DS Configuration

    1. Press and hold (Select), and while holding (Select), power on your console
    2. Your console should have booted into the Luma3DS configuration menu
      • Le menu de configuration de Luma3DS sert à régler les paramètres du custom firmware Luma3DS. Beaucoup de ces paramètres peuvent être utiles pour la personnalisation ou le débogage
      • Pour les besoins de ce guide, laissez ces options sur les paramètres par défaut (ne cochez ni ne décochez rien)
      • If your console shuts down when you try to power it on, ensure that you have copied boot.firm from the Luma3DS .zip to the root of your SD card
    3. Appuyez sur (Start) pour enregistrer les modifications et redémarrer

    Section VIII - Restoring DS Download Play

    1. Naviguez vers Nintendo 3DS -> <ID0> -> <ID1> -> Nintendo DSiWare sur votre carte SD
    2. Copiez le fichier 484E4441.bin du dossier clean de l’archive DSiWare téléchargée (output_(nom).zip) dans le dossier Nintendo DSiWare
    3. Réinsérez votre carte SD dans votre console
    4. Power on your console
    5. Launch System Settings on your console
    6. Naviguez versGestion des données -> DSiWare
    7. Sous la section “Carte SD”, sélectionnez le titre “Haxxxxxxxxx!”
    8. Sélectionnez “Copier”, puis sélectionnez “OK”
    9. Quittez Paramètres de la console
    10. Éteignez votre console
